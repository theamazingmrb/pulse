# Priority Compass Roadmap

## ✅ Completed

- [x] Rebrand to Priority Compass (PC)
- [x] Google Calendar integration (read)
- [x] Day/Week/Month calendar views
- [x] Click-to-add tasks from calendar
- [x] Polished landing page
- [x] Two-way Google Calendar sync (write scheduled tasks back)

## 🎯 Planned Features

### High Priority

- [ ] **Recurring Tasks** — Daily standups, weekly reviews, monthly reports. Set once, auto-create forever.
- [ ] **Focus Timer / Pomodoro** — 25-min focus sessions on specific tasks. Log focus time to journal. Pairs with Spotify.
- [ ] **Push Notifications** — Browser notifications for check-in reminders, task start times, overdue tasks, reflection reminders.
- [ ] **Quick Add (Cmd+K)** — Global keyboard shortcut to add task from anywhere. No context switching.

### Medium Priority

- [ ] **Productivity Analytics** — Dashboard showing tasks completed, focus time logged, check-in streaks, priority distribution, energy trends.
- [ ] **PWA / Offline Support** — Installable on mobile, queue tasks offline, sync when back online.
- [ ] **Habit Tracker** — Separate from tasks. Daily habits with streak tracking. Integrate with check-in flow.

### Future Consideration

- [ ] **AI-Suggested Scheduling** — Learn from check-in energy patterns, suggest optimal times, conflict resolution.
- [ ] **Data Export** — JSON/CSV export for journal entries, tasks, reflections.

## 📝 Notes

- Focus Timer is highest user value for effort ratio
- PWA would significantly improve mobile experience
- AI scheduling should wait until we have more usage data to learn from
